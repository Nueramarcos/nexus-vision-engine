# Contributing to Nexus Vision Engine

## Guidelines

1. **Python Version**: Ensure you are using Python 3.10 or later.
2. **Running the Demo**:
   - Navigate to the project directory.
   - Run the demo script by executing `python nexus/pipeline.py`.
   - This will start the pipeline and demonstrate its functionality.

## Contribution Notes

- If you encounter any issues, please open an issue on GitHub with a detailed description.
- For feature requests or improvements, feel free to submit a pull request.
- Ensure your code is well-documented and includes tests if applicable.
